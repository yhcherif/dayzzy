#dayzzy
---

Dayzzy python program that illustrate the ***Zeller's Congruence Algorithm***.
By taking any Julian or Gregorian Calendar date  in ""dd-mm-yyyy"" format, ***Dayzzy*** return the day of the week.

#Usage
---
To run dayzzy:

```
chmod +x dayzzy.py
./dayzzy.py
```
or 

`ptyhon dayzzy.py`
